# Relatum

A small graph-based note-taking app.

You can create notes, connect them, and navigate your ideas as a network instead of a linear list.

## Demo

You can try relatum here: [https://gr33b0k.github.io/relatum/](https://gr33b0k.github.io/relatum/)

## Features

- Notes with title, description and tags
- Connections between notes
- Search by title, tags and content
- Graph visualization (canvas)
- Data stored in localStorage

## Tech

- Vanilla JavaScript
- HTML / CSS
- Canvas
- localStorage

2. Open the project:
   ```sh

   ```
